# vue-xpack-starter
A vue webpack starter.
It has: Vue / Vuex / Vue-router / Webpack / ES6 / PostCSS + Stylus
> enjoy

## usage

```bash
# start
$ npm i
# develop
$ npm run dev
# for production
$ npm run build
```
